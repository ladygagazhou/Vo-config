欢迎来到 VOCAB LAB！这是一个纯本地化、极简高效且无干扰的英语学习实验室，旨在通过科学的交互设计帮你榨干每一个单词的记忆价值。

📖 核心功能指南
🗂️ 多学习计划：支持创建多个独立计划（如考研、雅思）。每个计划拥有完全隔离的词库、学习进度和顽固单词，可在“数据管理”中无缝切换与重命名。
🧘 沉浸模式：按下 F11 全屏时可一键开启。开启后系统将隐藏所有菜单、设置和页码，仅保留纯净的单词卡片，助你进入深度专注状态（按 Esc 退出）。
🔥 批量复习：在侧边栏点击「批量复习」，可将所有顽固单词瞬间搬运至首页进行突击强化。复习模式不影响正式计划进度，刷新页面或点击退出即可一键还原。
📖 导入词库：点击右上角「词库 / 导入」，支持内置词库或手动粘贴列表。系统支持带中文导入，也支持纯英文导入后自动联网抓取释义。
🎯 学习/复习模式：右上角一键切换。练习模式下会自动模糊释义与音标，鼠标悬停时才“刮开”显示，是自我测试与听写的绝佳工具。
💡 深度解释：将鼠标悬停在卡片右下角“灯泡”图标上 0.3 秒，系统将全网抓取发音、词性、近义词及真实的双语语境例句。
⭐ 顽固单词表：遇到难点点击“星号”收藏。通过侧边栏可随时查看列表，并支持针对顽固单词进行“翻卡测验”或“练习模式”强化。
📈 进度自动同步：系统实时记录每个计划的阅读轨迹。配合侧边栏的“学习进度”热力图，可随时点击页码回到当时的进度继续战斗。
🔊 循环发音与设置：可在顶部工具栏自由调整 列数/行数/字号/间距。支持单词循环朗读 1~5 次，并提供国内外多种自然语音引擎选择。
⌨️ 键盘快捷键：
← / →：快速翻页。
Alt + P / Ctrl + Alt + P：播放/停止本页单词朗读。
F11：进入全屏并触发沉浸模式询问。
Esc：退出沉浸模式或关闭所有弹窗。
🗄️ 数据安全：支持选择性导出/导入特定计划的数据项。建议定期在“数据管理”中导出备份，确保你的记忆宫殿永不丢失。
